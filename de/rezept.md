# Das Rezept

Bevor es um die Zutaten geht, muss eines klargestellt werden: Käsekuchen wird nicht aus Quark gemacht. Der Name des Kuchens ist Käsekuchen. Käse bedeutet Käse. Das Wort ist kein Ornament. Es ist eine Beschreibung. Ein Kuchen aus Quark ist ein Quarkkuchen, und dieser hat durchaus seine Berechtigung in der Welt, nur liegt diese Berechtigung woanders.

Die Füllung muss aus echtem Käse bestehen. Der Unterschied ist nicht subtil. Quark erzeugt eine Füllung, die leicht, mild und etwas luftig ist. Echter Käse erzeugt etwas Dichteres, Präsenteres, mit einem Geschmack, der nach dem Biss anhält. So soll sich ein Käsekuchen anfühlen.


## Zum Boden

Der ideale Käsekuchen hat keinen Boden.

Das ist keine radikale Position, auch wenn sie so behandelt wird. Der Boden in den meisten Käsekuchen existiert, weil Bäcker unsicher sind, ob die Füllung allein ausreicht. Es ist ein Selbstvertrauensproblem. Eine richtig gemachte Füllung braucht keine Unterlage, um ihre Existenz zu rechtfertigen. Sie kann direkt in einer gut vorbereiteten Springform gebacken, vollständig abgekühlt und ohne Zwischenfälle gestürzt werden.

Die Füllung ist der Käsekuchen. Der Boden ist Kommentar. Den Kommentar weglassen.

## Der Karottenkuchen-Boden

Wenn ein Boden wirklich gewünscht wird, ist ein Karottenkuchen-Boden die vertretbarste Option.

Nicht der vollständige Karottenkuchen mit Glasur und Schichten, nur der Boden: fein geriebene Karotten, gemahlene Mandeln, Öl, Ei, etwas Zucker und genug Struktur, um beim Backen zu halten. Das Ergebnis ist feucht, leicht nussig und bringt eine dezente Süße mit, die nicht mit der Füllung konkurriert. Ein Mürbeteig tendiert dazu, zu dominieren, sich anzukündigen, als Erstes wahrgenommen zu werden. Der Karottenkuchen-Boden macht das Gegenteil. Er ist da, wenn man ihn sucht, und nicht wahrnehmbar, wenn man es nicht tut. Das ist das richtige Verhalten für einen Boden.

Das ist eine persönliche Präferenz, keine Vorschrift. Die bodenlose Version bleibt die Referenzimplementierung.

```
    Käsekuchen    alternative Implementierung    Querschnitt

         ___________________________________
        |                                   |
        |         echter Käse               |
        |         Eier · Zucker · Vanille   |
        |         Zitronenschale            |
        |                                   |
        |===================================|   Karottenkuchen-Boden
        |   Karotten · Mandeln · Ei · Öl   |
        |___________________________________|

        der Boden ist vorhanden, kündigt sich aber nicht an
        das ist das richtige Verhalten für einen Boden
```


## Zutaten

Die Füllung benötigt Folgendes.

Echter Käse in ausreichender Menge als dominierende Zutat. Die genaue Sorte liegt im Ermessen des Bäckers, abhängig davon, was verfügbar ist und welches Geschmacksprofil angestrebt wird. Es muss echter Käse sein, kein Milchprodukt aus dem Regal neben dem Joghurt.

Eier. Die Eigelbe tragen zur Bindung und zur Farbe bei. Das Eiweiß kann separat steif geschlagen werden, um etwas Volumen einzubringen, oder als Ganzes untergerührt werden für ein dichteres Ergebnis. Beides ist richtig. Die Eier müssen vor der Verwendung Zimmertemperatur haben.

Zucker, abgestimmt auf den verwendeten Käse. Ein kräftigerer Käse braucht mehr, um die Balance zu finden. Ein milderer Käse braucht weniger. Eine universelle Grammzahl gibt es nicht, weil der Käse die Variable ist.

Vanille, von einer Vanilleschote oder als echtes Extrakt, nicht als synthetisches Aroma. Das ist eine der Zutaten, die im fertigen Ergebnis schmeckbar ist, ohne identifiziert zu werden. Ihre Abwesenheit fällt ebenfalls auf.

Zitronenschale von einer unbehandelten Zitrone. Die Schale hebt die Füllung an, ohne sie nach Zitrone schmecken zu lassen. Sie funktioniert als Aufheller für den Käsegeschmack, nicht als eigenständiges Aroma.

Ein Bindemittel: entweder Speisestärke oder Grieß. Eines von beidem. Nicht beides.

Schmand oder Sahne in kleiner Menge, wenn der verwendete Käse eher trocken ist. Das ist optional und sollte nach dem Kosten des rohen Käses entschieden werden.


## Was nicht hineinkommt

Obst kommt nicht in die Füllung. Es kommt nicht obendrauf. Es wird auch nicht als Beilage serviert.

Das ist keine Texturpräferenz. Obst zur Füllung hinzuzufügen ist ein implizites Misstrauensstatement gegenüber dem Käse. Es bedeutet, dass der Bäcker glaubte, der Käse allein sei nicht interessant genug, und versuchte, das mit Süße und Farbe aus einer anderen Quelle zu kompensieren. Wenn der Käse allein nicht interessant genug ist, lautet die Lösung, besseren Käse zu finden. Das Obst löst nichts und bringt Feuchtigkeit, Säure und visuelles Rauschen mit, um das die Füllung nicht gebeten hat.

Ein Käsekuchen, der mit einer Erdbeere obendrauf serviert wird, entschuldigt sich. Er sollte das nicht tun. Er sollte richtig sein.


## Backen und Kühlen

Im vorgeheizten Backofen bei einer Temperatur backen, die es der Füllung ermöglicht, vollständig durchzubacken, ohne oben aggressiv zu bräunen. Die Mitte sollte am Ende der Backzeit eine leichte Bewegung zeigen, wenn die Form sanft bewegt wird, ähnlich einem gesetzten Pudding. Sie festigt sich beim Abkühlen weiter.

Den Käsekuchen nicht sofort nach dem Backen aus dem Ofen nehmen. Den Ofen ausschalten und die Tür einen Spalt öffnen. Die Temperatur über dreißig bis vierzig Minuten langsam abfallen lassen, bevor die Form bewegt wird. Schnelle Temperaturwechsel lassen die Oberfläche reißen. Eine gerissene Oberfläche ist keine Katastrophe, aber sie ist vermeidbar.

Vollständig auf Zimmertemperatur abkühlen lassen, bevor der Kuchen in den Kühlschrank kommt. Mindestens vier Stunden kühlen vor dem Servieren. Über Nacht ist besser. Der Geschmack entwickelt sich in dieser Zeit, und die Textur wird das, was sie sein soll.

Kalt servieren. Nicht lauwarm. Nicht bei Zimmertemperatur. Kalt. Die Struktur der Füllung ist darauf ausgelegt, kalt erlebt zu werden.

```cuda
// der Backprozess, formalisiert
// Proteindenaturierung ist embarrassingly parallel
// jedes Molekül der Füllung setzt gleichzeitig
// das ist keine Metapher. so funktioniert Wärme.

__global__ void backen(
    float*      fuellung,
    const float ofen_celsius,    // 165.0f empfohlen, Umluft
    const int   dauer_minuten
) {
    int i = blockIdx.x * blockDim.x + threadIdx.x;

    fuellung[i] = denaturieren(fuellung[i], ofen_celsius, dauer_minuten);

    // alle Threads müssen abgeschlossen sein, bevor die Ofentür aufgeht
    // __syncthreads() ist kein Vorschlag
    __syncthreads();
}

// fuellung[i] nicht abfragen, während der Kernel läuft
// den Ofen nicht öffnen
// das Abkühlen ist ein separater Kernel, Geduld ist ein Pflichtparameter
// obst = nullptr    dieser Parameter existiert nicht
```
